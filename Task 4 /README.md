#  Sentiment Analysis

## Project Overview

This project analyzes customer reviews using Natural Language Processing (NLP) techniques to determine whether reviews are Positive, Negative, or Neutral.

The project demonstrates the complete Data Analytics workflow including:

- Data Collection
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sentiment Analysis
- Data Visualization
- Business Insights

---

## Dataset



Columns

- Review_ID
- Review
- Rating
- Sentiment

## Business Insights

- Positive Reviews: 62%
- Neutral Reviews: 15%
- Negative Reviews: 23%

Key Findings

- Customers appreciate product quality.
- Price is the most common complaint.
- Customer service can be improved.
