📚 Books Dataset - Exploratory Data Analysis (EDA)

Project Overview

This project demonstrates Exploratory Data Analysis (EDA) using Python, Pandas, and Matplotlib on a dataset containing 200 books.

The objective is to understand pricing, ratings, stock availability, and category distribution while generating business insights from the data.

---

Dataset

The dataset contains 200 books with the following columns:

- Book_ID
- Title
- Category
- Author
- Price
- Rating
- Stock

Categories include:

- Fiction
- Science
- History
- Technology
- Business
- Self-Help
- Programming
- AI
- Data Science

---

Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

---

Tasks Performed

1. Data Loading

Loaded the dataset using Pandas.

2. Dataset Exploration

- Shape
- Data Types
- Information
- Summary Statistics

3. Missing Value Analysis

Verified whether the dataset contains missing values.

4. Price Analysis

- Average Price
- Maximum Price
- Minimum Price

5. Rating Analysis

Calculated average ratings across different categories.

6. Category Analysis

Counted books in each category.

7. Top Expensive Books

Identified the top 10 most expensive books.

8. Outlier Detection

Used the IQR method to detect price outliers.

9. Correlation Analysis

Studied the relationship between Rating and Stock.

10. Hypothesis Testing

- Programming books are more expensive than Fiction books.
- Higher-rated books have higher stock levels.
