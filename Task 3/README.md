📚 Books Data Visualization Project

Overview

This project demonstrates how raw data can be transformed into meaningful visualizations using Python and Matplotlib.

The dataset contains information about 200 books, including category, author, price, rating, and stock availability. Through data visualization, the project uncovers trends and insights that can support business decisions.

---

Objectives

- Generate a dataset of 200 books.
- Explore and understand the dataset.
- Create meaningful charts and graphs.
- Present business insights through visual storytelling.
- Build a beginner-friendly Data Analyst portfolio project.

---

Dataset Columns

- Book_ID
- Title
- Category
- Author
- Price
- Rating
- Stock

---

Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

Visualizations

The project includes:

- Category-wise Book Count (Bar Chart)
- Average Price by Category
- Average Rating by Category
- Price Distribution (Histogram)
- Price Box Plot
- Rating vs Stock (Scatter Plot)
- Category Distribution (Pie Chart)

---

Business Insights

- Identified the category with the most books.
- Compared average prices across categories.
- Determined the highest-rated categories.
- Analyzed price distribution.
- Detected price outliers.
- Examined the relationship between rating and stock.
